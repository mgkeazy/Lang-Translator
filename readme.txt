For operating this project steps are given below

1. Install all libraries and packages as mentioned in package.json using command npm i in js directory
3. Rest of the things are working on client side by adding scrips so no need to worry about that
3. After doing all this start server for starting server go to js directory
 3.1 cd js
 3.2 node server.js
4. After starting server live out html file in vscode and then upload file which will take time some time beginning to extract text from image
5. All done
